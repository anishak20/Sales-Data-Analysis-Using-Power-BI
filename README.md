# Sales Data Analysis Dashboard (Power BI)

This project showcases an **interactive Power BI dashboard** developed to analyze store sales data and extract meaningful business insights through effective data modeling and visualization.

---

## Project Overview
The dashboard provides a comprehensive view of sales performance across products, customers, and regions. It allows users to explore trends, compare metrics, and gain actionable insights using interactive filters and visuals.

---

## Data Processing & Modeling
- Performed **data profiling** to understand data quality and structure  
- Applied **data cleaning and transformations** using **Power Query**  
- Designed a **star schema data model** for efficient analysis  
- Managed **relationships and cardinality** (one-to-many) between tables  
- Created **calculated columns and measures using DAX**

---

## Key Features
- Top/Bottom 5 products by **Sales, Profit, and Quantity Sold**  
- Sales trends over time (**daily, monthly, quarterly, yearly**)  
- Analysis of **Sales vs Profit relationship**  
- Comparison of **Sales, Profit, and Quantity** across user-selected periods  
- Average discount analysis by category  
- Total number of orders  
- Interactive order-level table with filters (**Product, Date, Customer, Promotion**)  
- **City-wise sales analysis**

---
